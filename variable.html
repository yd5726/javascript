이렇게 const로 배열이나

object 값을 추가하거나 삭제하는 것

즉 변경하는 것은 가능하다.

const const_array = ["apple", "orange","watermelon"];

const_array.push("banana");

console.log(const_array);

//['apple', 'orange', 'watermelon', 'banana']

immutable array(수정이 가능한 배열)

concat() 함수는 파라미터로 받은 배열이나 값들을 기존의 배열에 합쳐서, 새로운 배열을 만들어서 리턴한다.

const immutable1 = ['ab','bc'];

immutable2 = [].concat(immutable1, "cd");

console.log(immutable1); //['ab', 'bc']

console.log(immutable2); //['ab', 'bc', 'cd']

console.log(immutable1 === immutable2); //false
